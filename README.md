## Run Locally

Clone the project

```bash
  git clone https://github.com/siddharthraj09/test_Question_GIIT_Sol_Q1
```
Go to the project directory

```bash
  cd fetch
```

Install dependencies

```bash
  npm install node-fetch
```

Start the server

```bash
  node .\app.js
```
